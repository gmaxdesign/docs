---
title: Mark specific messages as read for a given instance.
---

```http
PUT /chat/markMessageAsRead/{instanceName}
```

This endpoint allows users to mark messages as read for a particular instance.

### Parâmetros

- `instanceName` (path, obrigatório) — The name of the instance to which the reaction should be sent.

### Respostas
#### 200
Successful response
