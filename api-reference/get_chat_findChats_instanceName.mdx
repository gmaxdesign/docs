---
title: List all chats associated with a specific instance.
---

```http
GET /chat/findChats/{instanceName}
```

This endpoint retrieves a list of all chats associated with the specified instance.

### Parâmetros

- `instanceName` (path, obrigatório) — The name of the instance to which the reaction should be sent.

### Respostas
#### 200
Successful response
