---
title: Convert media message content to Base64.
---

```http
POST /chat/getBase64FromMediaMessage/{instanceName}
```

This endpoint retrieves the Base64 representation of the content of a media message for the specified instance.

### Parâmetros

- `instanceName` (path, obrigatório) — The name of the instance to which the reaction should be sent.

### Respostas
#### 200
Successful response
