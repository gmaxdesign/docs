---
title: Provide a list of WhatsApp numbers associated with a given instance.
---

```http
POST /chat/whatsappNumbers/{instanceName}
```

This endpoint returns information on the WhatsApp numbers associated with the specified instance.

### Parâmetros

- `instanceName` (path, obrigatório) — The name of the instance to which the reaction should be sent.

### Respostas
#### 200
Successful response
