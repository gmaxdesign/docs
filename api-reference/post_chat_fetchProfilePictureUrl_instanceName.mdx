---
title: Retrieve the profile picture URL of a specific number.
---

```http
POST /chat/fetchProfilePictureUrl/{instanceName}
```

This endpoint fetches the profile picture URL associated with the given phone number for the specified instance.

### Parâmetros

- `instanceName` (path, obrigatório) — The name of the instance to which the reaction should be sent.

### Respostas
#### 200
Successful response
