---
title: Send an audio message via WhatsApp to a specified instance.
---

```http
POST /message/sendWhatsAppAudio/{instanceName}
```

This endpoint allows users to share an audio message.

### Parâmetros

- `instanceName` (path, obrigatório) — The name of the instance to which the audio should be sent.

### Respostas
#### 200
Successful response
