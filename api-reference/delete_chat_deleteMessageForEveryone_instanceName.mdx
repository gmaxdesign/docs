---
title: Delete a message for everyone in a given instance.
---

```http
DELETE /chat/deleteMessageForEveryone/{instanceName}
```

This endpoint allows users to delete a message for everyone in the chat.

### Parâmetros

- `instanceName` (path, obrigatório) — The name of the instance to which the reaction should be sent.

### Respostas
#### 200
Successful response
